# --REQUIREMENTS --

	Joomla 2.5;
	HikaShop 1.6;

# -- INSTRUCTION --

	1. Copy paysera directory to Joomla tmp directory;
	2. Go to extensions->extension manager->install;
	3. On "Install from directory" area click "Install" button;
	4. Go to extensions->extension manager->manage and enable "HikaShop Paysera.com payment plugin";
	5. Go to Components->Hikashop->System->Payment methods->HikaShop Paysera.com payment plugin and
	write required info. Press "Save and close" button;
